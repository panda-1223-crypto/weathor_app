■ 概要

このアプリは、検索ボックスに入力した地域の天気予報を外部 API から取得し、

画面に表示するシンプルな天気検索ツールです。

お気に入り登録や検索履歴にも対応しており、よく調べる地域を素早く呼び出すことができます。



■　使用技術

・JavaScript

・HTML/CSS



■　主な機能

・天気検索機能（APIへリクエスト）

・お気に入り（★）機能

・検索履歴

・削除機能



■　起動方法

・インストール作業やコマンド操作は不要です。

・このリポジトリをダウンロード or クローン

・index.html` をダブルクリックしてブラウザで開くだけで実行可能！
